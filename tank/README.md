## 概述

Tank 90 是一款基于 HTML5 Canvas + TypeScript 开发的经典坦克大战游戏，复刻自 90 年代烟山版坦克大战。游戏采用像素风格程序化绘制，无需任何外部图片资源。

## 特性

- **经典玩法**: 保卫基地，消灭敌方坦克
- **7 种地形**: 空地、砖墙、钢墙、森林、水面、冰面、基地
- **4 种敌方坦克**: 普通、快速、强力、重甲
- **7 种道具**: 星星、1UP、铲子、头盔、时钟、炸弹、船
- **道具反制**: 敌方也可拾取道具触发对玩家不利的效果（烟山版特性）
- **键盘 + 鼠标双操作**: 支持 WASD/方向键移动，也支持鼠标拖拽和点击射击
- **程序化音效**: 使用 Web Audio API 合成所有音效
- **粒子特效**: 坦克爆炸、子弹撞击、道具拾取
- **20 个精心设计的关卡**

## 操作指南

### 键盘控制

| 动作 | 按键 |
|------|------|
| 移动 | W/A/S/D 或 方向键 |
| 射击 | Space 或 Z |
| 暂停 | P 或 Escape |
| 确认 | Enter 或 Space |

### 鼠标控制

- **拖拽移动**: 在坦克区域内按下左键并拖拽，坦克向拖拽方向移动
- **点击射击**: 在坦克区域外单击左键转向并射击
- **持续射击**: 在坦克区域外保持按下左键连续发射
- **右键射击**: 向当前朝向射击

## 游戏规则

### 胜利条件
- 消灭本关所有敌方坦克

### 失败条件
- 基地被摧毁
- 玩家生命耗尽

### 计分

| 消灭目标 | 分值 |
|---------|------|
| 普通坦克 | 100 |
| 快速坦克 | 200 |
| 强力坦克 | 300 |
| 重甲坦克 | 400 |

每 20000 分奖励 1 条命。

## 地图规格

- **画布**: 600 × 800 像素
- **HUD**: 600 × 40 像素（顶部）
- **战场**: 600 × 760 像素
- **网格**: 30 列 × 38 行
- **单元格**: 20 × 20 像素
- **坦克**: 40 × 40 像素（2×2 格子）

## 坐标系

游戏使用战场坐标系，原点 `(0, 0)` 为战场左上角（屏幕坐标 `(0, 40)`）。

- 实体位置: `(x, y)` 战场像素坐标
- 碰撞检测: 在战场坐标系内进行
- 渲染时: `screenY = entity.y + BATTLE_AREA_Y` (40px)

## 敌方 AI

1. 与玩家同行/列且无障碍 → 射击
2. 与玩家同行/列 → 转向接近
3. 地图上有道具且距离<8格 → 贪心导航
4. 被阻挡 3 帧 → 随机转向
5. 默认：沿当前方向移动，每 180 帧 30% 概率随机换向

## 开发指南

### 添加新关卡

在 `src/world/levels/` 目录下创建 `levelXX.txt` 文件，格式为 38 行 × 30 列的数字，每行数字用空格分隔：

- `0`: 空地
- `1`: 砖墙
- `2`: 钢墙
- `3`: 森林
- `4`: 水面
- `5`: 冰面

```bash
npm run lint
```
