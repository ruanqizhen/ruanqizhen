# HTML5 Galaga (小蜜蜂) 复刻增强版

[体验地址](https://qizhen.xyz/galaga/)

这是一个基于 HTML5 Canvas 和 原生 JavaScript 开发的经典“小蜜蜂 (Galaga)”复刻射击游戏。在保留了原版街机经典阵型玩法的基础上，进行了玩法升级，引入了护盾、精英怪等机制。

## 游戏预览

玩家操控底部的战机，抵御源源不断从屏幕上方涌入并组成阵型的外星敌机。除了普通的无人机和拦截机，后期的关卡将面临拥有实弹追踪制导武器的精英护盾敌机，甚至体型庞大且具有多个独立破坏部位的史诗级 Boss 战舰。

## 🌟 模块化架构

- `config.js`: 游戏全局常量、状态配置池。
- `audio.js`: 音频上下文调度系统，实现跨平台的无缝音效播放。
- `effects.js`: 粒子系统模块，负责爆炸、护盾碎裂等视觉特效呈现。
- `pools.js`: 对象池机制，专门用于循环利用被销毁的子弹和粒子实体，大幅降低垃圾回收引起的卡顿。
- `entities.js`: 面向对象实战系统，定义了 `Player`（玩家）、`Enemy`（敌机）、`Boss`（头目）、`TrackingMissile`（追踪导弹）等核心游戏实体。
- `collision.js`: 独立的分发碰撞中心，管理实体交汇并处理逻辑反馈。
- `game.js`: 主游戏循环引擎，调度 `requestAnimationFrame` 和输入捕获。



## 🎮 控制指南

- **键盘操作**：
  - `左/右方向键 (Left/Right Arrow)`：向左 / 向右移动战机
  - `空格键 (Space)`：发射主武器，按住可实现高速连发机制（配合极速 Buff 效果更佳）
- **触控/鼠标操作**：
  - 支持直接 `拖拽` 战机左右滑动游玩，并在拖拽过程中自动开火，对移动端优化极度友好。

## 🛠️ 安装与运行

本项目为纯前端 HTML5 原生实现，没有任何后端依赖。

1. **直接点开**项目的 `index.html` 文件，即可在任何现代浏览器中畅玩。
2. 或**一键启动本地终端服务**（推荐）：
   如果在开发环境下由于跨域限制导致外部图片或音效加载报错，可以使用 Python 等工具开启本地服务：
   ```bash
   python -m http.server 8000
   # 或者使用 Node.js 工具：
   npx serve .
   ```
   随后在浏览器中访问 `http://localhost:8000` 即可享有最高效的性能体验！
